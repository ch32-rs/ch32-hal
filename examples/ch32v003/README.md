# ch32v003-examples

## Boards

### nanoCH32V003

Link: [wuxx/nanoCH32V003](https://github.com/wuxx/nanoCH32V003)

- MCU: CH32V003F4U6
- LED: PD6

### LineDog CH32V003J4M6 Board

Link: [OSHWHUB](https://oshwhub.com/andelf/linedog)

- MCU: CH32V003J4M6
- LED: PA2, PC4
- No HSE crystal

### LineDog v2 CH32V003F4P6 Board


- BUZZ: PC4 (TIM1CH4)
- WS2812: PA2
